Te BedBooking widget allows you to sell accomodation through your website. The widget automatically updates available dates and bookings based on the data from the BedBooking application.

== Description ==

You will receive information about all bookings made with the widget via the mobile application, emial or WAP push message service.

This plugin communicates with our api. So it need username and password form our system.

Widget calls
https://panel.bed-booking.com/api/getwserwidgetoptions to get user rooms from our system. Those are listed in dropdown and then displayed on fron end.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to BedBooking menu page.
4. Register, or login to our system by providing username(email) and password. The same You use to login at https://panel.bed-booking.com/index/login
5. Go to widgets.
6. Drag BedBooking Widget to sidebar
7. Choose type and language of calendar
8. Enjoy!

== Frequently asked questions ==

= I don't have account, where can i register? =

To an account on bedbooking, you need to download out mobile app from appstore(https://itunes.apple.com/US/app/id826280602?mt=8),google play (https://play.google.com/store/apps/details?id=com.rst.bedbooking) or windows store (https://www.windowsphone.com/pl-pl/store/app/bedbooking/7f2561b5-474a-41bd-9153-08cf963dd39c). Register new account, and you will receive username and password.
