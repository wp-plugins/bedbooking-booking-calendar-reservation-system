Booking calendar & booking system for availability checking and online reservations. Widget automatically sync bookings with mobile/desktop app.

== Description ==

Booking calendar plugin allows you to sell accommodation through your website. The plugin enables online reservations via PayPal.

 

Plugin works perfectly for apartments, agrotourism, private accommodation, B&B hotels, hostels, boarding houses, guest rooms, inns, holiday resorts, villas, and accommodation bureau management.

 

This plugin automatically updates available dates and bookings based on the data from the BedBooking application. You will receive information about all bookings made with the plugin via the push message service. The plugin is a part of booking system.

 

BedBooking users have already made more than 150,000 bookings! Download this plugin and find out how easily you can manage your overnight accommodation.

**More options:**

**BOOKING MANAGEMENT**

Manage your overnight accommodation booking from your mobile phone or tablet. Wherever you are. With BedBooking installed on your phone, you can quickly check vacancies and make a booking for a customer during a call. On the basis of the date and number of people, the application will automatically calculate the amount due, thanks to which you can quickly tell the customer about the price.

 

**FAST AND EASY SAVINGS**

Thanks to the transparent calendar, you can easily check vacancies for a given period and then with one click make the booking. The application automatically links the phone number with the booking, while use of speaker mode allows the information to be saved during the call.

 

**BOOKINGS CALENDAR SHARING**

The BedBooking application enables simultaneous access to one account with the calendar using a number of different devices. Each of your co-workers can follow the current occupancy rates and make bookings with their own phones. BedBooking provides automatic security against conflicts and overlapping booking dates.

 

**ACCESS VIA COMPUTER**

BedBooking enables management of bookings via computer after logging into the website: <a href="http://bed-booking.com" target="_blank" title="http://bed-booking.com">http://bed-booking.com</a> To log in to your BedBooking account, use the login and password saved in the BedBooking mobile application settings. On the website you will be able to configure the price list. You will also find your customers’ database and bookings and occupancy statistics.

 

**DATA SAFETY**

Data about bookings are saved on our server, which enables the automatic creation of a backup copy. In case of mobile phone or tablet loss or theft, you can change the password to access your bookings calendar. Access to your BedBooking account is protected and requires authorisation in the form of a login and password.

 

**BOOKING SYSTEM**

BedBooking enables the creation of a searcher with numerous overnight accommodation facilities in a very simple way.

== Installation ==

**Short description:**

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

 

**Long description:**

1. Upload \`plugin-name.php\` to the \`/wp-content/plugins/\` directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Go to BedBooking menu page.

4. Register or login to BedBooking system by providing username (email) and password. 

5. Go to widgets.

6. Drag the BedBooking widget to sidebar.

7. Choose a type and language of booking calendar.

8. Enjoy!
